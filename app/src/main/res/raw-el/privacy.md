## Εισαγωγή

Το File Explorer δεσμεύεται να προστατεύει το απόρρητό σας. Αυτή η Πολιτική Απορρήτου εξηγεί πώς η εφαρμογή χειρίζεται τις πληροφορίες σας.

## Συλλογή Δεδομένων

**Το File Explorer δεν συλλέγει, αποθηκεύει ή μεταδίδει προσωπικά δεδομένα.** Η εφαρμογή λειτουργεί εξ ολοκλήρου στη συσκευή σας και δεν επικοινωνεί με εξωτερικούς διακομιστές για σκοπούς συλλογής δεδομένων.

## Πρόσβαση σε Αρχεία

Το File Explorer απαιτεί πρόσβαση στον αποθηκευτικό χώρο της συσκευής σας για να εκτελέσει τις βασικές του λειτουργίες:

- **Περιήγηση σε αρχεία και φακέλους**
- **Αντιγραφή, μετακίνηση και διαγραφή αρχείων**
- **Μετονομασία αρχείων και φακέλων**
- **Προβολή πληροφοριών αρχείων και μεταδεδομένων**

Όλες οι λειτουργίες αρχείων εκτελούνται τοπικά στη συσκευή σας. Κανένα δεδομένο αρχείου δεν μεταδίδεται εξωτερικά.

## Δικαιώματα

Η εφαρμογή ζητά τα ακόλουθα δικαιώματα:

- **Πρόσβαση αποθήκευσης**: Απαιτείται για περιήγηση, διαχείριση και οργάνωση των αρχείων σας
- **Πρόσβαση σε όλα τα αρχεία (Android 11+)**: Απαιτείται για πρόσβαση σε αρχεία σε όλες τις τοποθεσίες αποθήκευσης

Αυτά τα δικαιώματα χρησιμοποιούνται αποκλειστικά για τη λειτουργικότητα διαχείρισης αρχείων εντός της εφαρμογής.

## Υπηρεσίες Τρίτων

Το File Explorer χρησιμοποιεί τις ακόλουθες υπηρεσίες τρίτων:

- **Firebase Crashlytics**: Συλλέγει ανώνυμες αναφορές σφαλμάτων για τη βελτίωση της σταθερότητας της εφαρμογής. Καμία προσωπική πληροφορία δεν περιλαμβάνεται σε αυτές τις αναφορές.
- **Firebase Analytics**: Συλλέγει ανώνυμα στατιστικά χρήσης για τη βελτίωση της εμπειρίας της εφαρμογής. Δεν συλλέγονται προσωπικά αναγνωρίσιμες πληροφορίες.

## Ασφάλεια Δεδομένων

Δεδομένου ότι όλα τα δεδομένα σας παραμένουν στη συσκευή σας, προστατεύονται από τα μέτρα ασφαλείας της συσκευής σας, συμπεριλαμβανομένου του κλειδώματος οθόνης και των ρυθμίσεων κρυπτογράφησης.

## Απόρρητο Παιδιών

Αυτή η εφαρμογή δεν προορίζεται για χρήση από παιδιά κάτω των 13 ετών. Συνιστάται γονική επίβλεψη για νεότερους χρήστες.

## Αλλαγές σε Αυτή την Πολιτική

Ενδέχεται να ενημερώνουμε αυτή την Πολιτική Απορρήτου κατά καιρούς. Οποιεσδήποτε αλλαγές θα αντικατοπτρίζονται στην εφαρμογή με ενημερωμένη ημερομηνία αναθεώρησης.
