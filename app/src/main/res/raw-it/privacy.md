## Introduzione

File Explorer si impegna a proteggere la tua privacy. Questa Informativa sulla Privacy spiega come l'applicazione gestisce le tue informazioni.

## Raccolta dei dati

**File Explorer non raccoglie, memorizza o trasmette alcun dato personale.** L'app funziona interamente sul tuo dispositivo e non comunica con server esterni per scopi di raccolta dati.

## Accesso ai file

File Explorer richiede l'accesso allo storage del tuo dispositivo per svolgere le sue funzioni principali:

- **Navigazione tra file e cartelle**
- **Copia, spostamento ed eliminazione di file**
- **Rinomina di file e cartelle**
- **Visualizzazione di informazioni e metadati dei file**

Tutte le operazioni sui file vengono eseguite localmente sul tuo dispositivo. Nessun dato dei file viene trasmesso all'esterno.

## Autorizzazioni

L'app richiede le seguenti autorizzazioni:

- **Accesso allo storage**: Necessario per navigare, gestire e organizzare i tuoi file
- **Accesso a tutti i file (Android 11+)**: Necessario per accedere ai file in tutte le posizioni di archiviazione

Queste autorizzazioni vengono utilizzate esclusivamente per le funzionalità di gestione dei file all'interno dell'app.

## Servizi di terze parti

File Explorer utilizza i seguenti servizi di terze parti:

- **Firebase Crashlytics**: Raccoglie report anonimi sugli arresti anomali per aiutare a migliorare la stabilità dell'app. Nessuna informazione personale è inclusa in questi report.
- **Firebase Analytics**: Raccoglie statistiche anonime sull'utilizzo per aiutare a migliorare l'esperienza dell'app. Non vengono raccolte informazioni di identificazione personale.

## Sicurezza dei dati

Poiché tutti i tuoi dati rimangono sul tuo dispositivo, sono protetti dalle misure di sicurezza del tuo dispositivo, inclusi il blocco schermo e le impostazioni di crittografia.

## Privacy dei minori

Questa app non è destinata all'uso da parte di bambini di età inferiore ai 13 anni. Si raccomanda la supervisione dei genitori per gli utenti più giovani.

## Modifiche a questa informativa

Potremmo aggiornare questa Informativa sulla Privacy di tanto in tanto. Qualsiasi modifica sarà riflessa nell'app con una data di revisione aggiornata.
