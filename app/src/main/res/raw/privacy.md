## Introduction

File Explorer is committed to protecting your privacy. This Privacy Policy explains how the application handles your information.

## Data Collection

**File Explorer does not collect, store, or transmit any personal data.** The app operates entirely on your device and does not communicate with external servers for data collection purposes.

## File Access

File Explorer requires access to your device's storage to perform its core functions:

- **Browsing files and folders**
- **Copying, moving, and deleting files**
- **Renaming files and folders**
- **Viewing file information and metadata**

All file operations are performed locally on your device. No file data is transmitted externally.

## Permissions

The app requests the following permissions:

- **Storage Access**: Required to browse, manage, and organize your files
- **All Files Access (Android 11+)**: Required to access files across all storage locations

These permissions are used solely for file management functionality within the app.

## Third-Party Services

File Explorer uses the following third-party services:

- **Firebase Crashlytics**: Collects anonymous crash reports to help improve app stability. No personal information is included in these reports.
- **Firebase Analytics**: Collects anonymous usage statistics to help improve the app experience. No personally identifiable information is collected.

## Data Security

Since all your data remains on your device, it is protected by your device's security measures, including screen lock and encryption settings.

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the app with an updated revision date.

## Contact

If you have questions about this Privacy Policy, please contact us through the app's feedback feature.
