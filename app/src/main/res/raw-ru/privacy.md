## Введение

File Explorer стремится защитить вашу конфиденциальность. Настоящая Политика конфиденциальности объясняет, как приложение обрабатывает вашу информацию.

## Сбор данных

**File Explorer не собирает, не хранит и не передаёт персональные данные.** Приложение работает полностью на вашем устройстве и не связывается с внешними серверами для сбора данных.

## Доступ к файлам

File Explorer требует доступ к хранилищу вашего устройства для выполнения своих основных функций:

- **Просмотр файлов и папок**
- **Копирование, перемещение и удаление файлов**
- **Переименование файлов и папок**
- **Просмотр информации о файлах и метаданных**

Все операции с файлами выполняются локально на вашем устройстве. Никакие данные файлов не передаются вовне.

## Разрешения

Приложение запрашивает следующие разрешения:

- **Доступ к хранилищу**: Необходим для просмотра, управления и организации ваших файлов
- **Доступ ко всем файлам (Android 11+)**: Необходим для доступа к файлам во всех местах хранения

Эти разрешения используются исключительно для функций управления файлами в приложении.

## Сторонние сервисы

File Explorer использует следующие сторонние сервисы:

- **Firebase Crashlytics**: Собирает анонимные отчёты о сбоях для улучшения стабильности приложения. В эти отчёты не включается персональная информация.
- **Firebase Analytics**: Собирает анонимную статистику использования для улучшения работы приложения. Персональные идентифицирующие данные не собираются.

## Безопасность данных

Поскольку все ваши данные остаются на вашем устройстве, они защищены мерами безопасности вашего устройства, включая блокировку экрана и настройки шифрования.

## Конфиденциальность детей

Это приложение не предназначено для использования детьми младше 13 лет. Для младших пользователей рекомендуется родительский контроль.

## Изменения в политике

Мы можем время от времени обновлять эту Политику конфиденциальности. Любые изменения будут отражены в приложении с обновлённой датой редакции.
