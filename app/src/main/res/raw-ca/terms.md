## Acceptació dels termes

En descarregar, instal·lar o utilitzar File Explorer, accepteu quedar vinculats per aquests Termes i Condicions. Si no esteu d'acord amb aquests termes, si us plau no utilitzeu l'aplicació.

## Llicència

File Explorer us atorga una llicència limitada, no exclusiva i intransferible per utilitzar l'aplicació al vostre dispositiu personal per a finalitats personals o comercials.

## Ús permès

Podeu utilitzar File Explorer per a:

- Navegar i explorar el sistema de fitxers del vostre dispositiu
- Copiar, moure, canviar el nom i eliminar fitxers i carpetes
- Veure informació i metadades dels fitxers
- Compartir fitxers amb altres aplicacions

## Ús prohibit

Accepteu no:

- Utilitzar l'aplicació per a cap finalitat il·legal
- Intentar fer enginyeria inversa, descompilar o desmuntar l'aplicació
- Eliminar o modificar qualsevol avís o etiqueta de propietat
- Utilitzar l'aplicació per accedir a fitxers als quals no teniu permís d'accés

## Renúncia de garanties

File Explorer es proporciona "tal qual" sense garanties de cap tipus, ja siguin expresses o implícites. No garantim que:

- L'aplicació complirà els vostres requisits específics
- L'aplicació serà ininterrompuda, puntual o lliure d'errors
- Els resultats obtinguts de l'ús de l'aplicació seran precisos o fiables

- Pèrdua de dades o fitxers

**Important**: Mantingueu sempre còpies de seguretat dels vostres fitxers importants. Les operacions amb fitxers com l'eliminació són permanents i no es poden desfer per l'aplicació.

Sou l'únic responsable de:

- Mantenir còpies de seguretat de les vostres dades importants
- Assegurar-vos que teniu els permisos adequats per gestionar fitxers
