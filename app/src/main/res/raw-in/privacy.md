## Pendahuluan

File Explorer berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana aplikasi menangani informasi Anda.

## Pengumpulan Data

**File Explorer tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi apa pun.** Aplikasi beroperasi sepenuhnya di perangkat Anda dan tidak berkomunikasi dengan server eksternal untuk tujuan pengumpulan data.

## Akses File

File Explorer memerlukan akses ke penyimpanan perangkat Anda untuk menjalankan fungsi intinya:

- **Menjelajah file dan folder**
- **Menyalin, memindahkan, dan menghapus file**
- **Mengganti nama file dan folder**
- **Melihat informasi dan metadata file**

Semua operasi file dilakukan secara lokal di perangkat Anda. Tidak ada data file yang dikirimkan secara eksternal.

## Izin

Aplikasi meminta izin berikut:

- **Akses Penyimpanan**: Diperlukan untuk menjelajah, mengelola, dan mengatur file Anda
- **Akses Semua File (Android 11+)**: Diperlukan untuk mengakses file di semua lokasi penyimpanan

Izin ini digunakan semata-mata untuk fungsi manajemen file dalam aplikasi.

## Layanan Pihak Ketiga

File Explorer menggunakan layanan pihak ketiga berikut:

- **Firebase Crashlytics**: Mengumpulkan laporan kerusakan anonim untuk membantu meningkatkan stabilitas aplikasi. Tidak ada informasi pribadi yang disertakan dalam laporan ini.
- **Firebase Analytics**: Mengumpulkan statistik penggunaan anonim untuk membantu meningkatkan pengalaman aplikasi. Tidak ada informasi yang dapat mengidentifikasi pribadi yang dikumpulkan.

## Keamanan Data

Karena semua data Anda tetap berada di perangkat Anda, data tersebut dilindungi oleh langkah-langkah keamanan perangkat Anda, termasuk kunci layar dan pengaturan enkripsi.
