## Penerimaan Syarat

Dengan mengunduh, menginstal, atau menggunakan File Explorer, Anda setuju untuk terikat oleh Syarat dan Ketentuan ini. Jika Anda tidak setuju dengan syarat-syarat ini, harap jangan gunakan aplikasi.

## Lisensi

File Explorer memberi Anda lisensi terbatas, non-eksklusif, dan tidak dapat dipindahtangankan untuk menggunakan aplikasi di perangkat pribadi Anda untuk tujuan pribadi atau komersial.

## Penggunaan yang Diizinkan

Anda dapat menggunakan File Explorer untuk:

- Menjelajah dan menavigasi sistem file perangkat Anda
- Menyalin, memindahkan, mengganti nama, dan menghapus file dan folder
- Melihat informasi dan metadata file
- Berbagi file dengan aplikasi lain

## Penggunaan yang Dilarang

Anda setuju untuk tidak:

- Menggunakan aplikasi untuk tujuan ilegal apa pun
- Mencoba merekayasa balik, mendekompilasi, atau membongkar aplikasi
- Menghapus atau memodifikasi pemberitahuan atau label kepemilikan apa pun
- Menggunakan aplikasi untuk mengakses file yang tidak Anda miliki izin untuk mengaksesnya

## Penafian Jaminan

File Explorer disediakan "sebagaimana adanya" tanpa jaminan apa pun, baik tersurat maupun tersirat. Kami tidak menjamin bahwa:

- Aplikasi akan memenuhi kebutuhan spesifik Anda
- Aplikasi akan tidak terputus, tepat waktu, atau bebas dari kesalahan
- Hasil yang diperoleh dari penggunaan aplikasi akan akurat atau dapat diandalkan

## Batasan Tanggung Jawab

Sejauh diizinkan oleh hukum, kami tidak bertanggung jawab atas kerusakan tidak langsung, insidental, khusus, konsekuensial, atau punitif, termasuk namun tidak terbatas pada:

- Kehilangan data atau file
- Kehilangan keuntungan atau pendapatan
- Kerusakan pada perangkat Anda
- Kerugian lain yang timbul dari penggunaan aplikasi ini

**Penting**: Selalu simpan cadangan file penting Anda. Operasi file seperti penghapusan bersifat permanen dan tidak dapat dibatalkan oleh aplikasi.

## Tanggung Jawab Pengguna

Anda bertanggung jawab penuh untuk:

- Menjaga cadangan data penting Anda
- Memastikan Anda memiliki izin yang sesuai untuk mengelola file
- Setiap konsekuensi yang dihasilkan dari operasi file yang Anda lakukan

## Modifikasi

Kami berhak untuk memodifikasi Syarat dan Ketentuan ini kapan saja. Penggunaan aplikasi yang berkelanjutan setelah perubahan merupakan penerimaan terhadap syarat yang dimodifikasi.

## Hukum yang Berlaku

Syarat dan Ketentuan ini akan diatur dan ditafsirkan sesuai dengan hukum yang berlaku, tanpa memperhatikan ketentuan konflik hukum.
